Trang giải thích Internet và Web

Internet là một mạng lưới toàn cầu kết nối hàng triệu máy tính và thiết bị với nhau, cho phép truyền tải dữ liệu qua các giao thức như TCP/IP.

Web (World Wide Web) là một dịch vụ chạy trên Internet, cho phép người dùng truy cập các trang web thông qua trình duyệt bằng giao thức HTTP/HTTPS.

  • IP Address: Địa chỉ định danh duy nhất của mỗi thiết bị trên mạng
  • HTTP/HTTPS: Giao thức truyền tải siêu văn bản dùng để tải trang web
  • DNS: Hệ thống phân giải tên miền, chuyển đổi tên miền thành địa chỉ IP
  • HTML: Ngôn ngữ đánh dấu siêu văn bản dùng để tạo cấu trúc trang web
